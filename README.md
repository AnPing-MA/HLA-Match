# OpenHLA Atlas

### Explain. Integrate. Interpret.

**An open knowledge hub for HLA matching, population genetics, and immunogenetics.**

---

## Why OpenHLA Atlas?

The HLA ecosystem is rich—but fragmented.

Researchers often need to consult multiple independent resources, including official databases, scientific publications, clinical guidelines, and bioinformatics tools, to answer a single HLA-related question.

OpenHLA Atlas aims to bridge these resources by helping users:

- **Explain** complex HLA concepts
- **Integrate** knowledge from authoritative sources
- **Interpret** HLA information in practical research and clinical contexts

This project is **not another HLA database**.

Instead, it serves as an open knowledge hub that helps researchers understand **what to use, when to use it, and why it matters.**
